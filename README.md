# demo-hotel-booking
This a Demo using LangGraph to Booking a Hotel Room based on ReAct architecture.

Create a virtual environment and activate it:

```bash
python3 -m venv langchain-course-env
source ./langchain-course-env/bin/activate
pip install -r ./requirements.txt
```

Create an .env file with the following information using the account IBM Watsonx:
```bash
API_KEY=xyz
PROJECT_ID=xyz
WATSONX_AI_URL=https://us-south.ml.cloud.ibm.com
```
